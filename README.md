🏦Simple Python ATM Simulator
A lightweight, object-oriented terminal application that simulates basic bank teller machine functionalities. Built with Python, this project demonstrates core programming concepts like classes, authentication logic, and state management.

✨ Features
🔒 Secure Login: Simple 4-digit PIN authentication with a maximum of 3 attempts.

💰 Balance Management: Real-time tracking of funds.

💵 Transactions: * Deposit: Add funds to your account.

Withdraw: Deduct funds (includes logic to prevent overdrafts).

📜 Transaction History: View a log of all activities performed during the session.
